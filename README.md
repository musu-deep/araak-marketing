# Araak Marketing Platform

منصة أراك لإدارة المناقصات وتسويق المشاريع هي منظومة رقمية تنفيذية متكاملة صُممت لخدمة مجموعة أراك في إدارة دورة حياة الفرص والمناقصات والمشاريع، من مرحلة الرصد والاستكشاف وحتى الإسناد والمتابعة والتقارير التنفيذية.

## الهدف

تحويل إدارة الفرص والمناقصات من عمل متفرق إلى منصة موحدة تساعد الإدارة العليا وفرق التسويق والمشاريع على:

* رصد الفرص والمناقصات.
* تقييم جاذبية الفرص واحتمالية الفوز.
* إدارة مراحل المناقصة.
* إسناد المهام لأعضاء الفريق.
* متابعة المسؤوليات والمساءلة.
* إدارة الوثائق والدروس المستفادة.
* إنتاج تقارير تنفيذية.
* تمكين صلاحيات مرنة حسب الأدوار.
